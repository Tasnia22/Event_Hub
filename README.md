# 🎉 Event Hub

A full-stack event management platform that allows users to discover, create, and book events seamlessly.

---

## ✨ Features

- 🔐 User Authentication
- 🎫 Event Booking
- 🗓️ Event Management
- 👤 User Dashboard
- 📱 Responsive Design
- 🔎 Search & Filter Events

---

## 🛠 Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js

### Database
- MongoDB

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/Tasnia22/Event_Hub.git
```

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
